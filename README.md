***Firebase*** 
***Firebase Firestore*** 
***Firebase Authentication*** 
***React JS***
***React Context API***
***React Reducers***

Download the code & run **npm install** or **npm i** to install dependencies before starting the app.

For Starting the app run **npm run start**.
